낚시 비법 (Angler's Secret)

Balatro × Slay the Spire × 낚시 — 카드 기반 로그라이크 낚시 게임

"한 번의 출조(Run)는 15~20분. 매번 다른 낚시터 경로를 골라 나아가고, 잡은 물고기로 얻은 '낚시 비법'을 키우고 조합해 점수를 폭발시킨다. 실패하면 즉시 끝나지만, 도감과 명성은 영구히 남는다."

🎣 지금 플레이하기 (Play Now)

🎮 게임 특징

  • 2층 구조 게임 디자인 — 한 판(15~20분) 안에 끝나는 RUN + 영구 진행(META) 시스템
  • Slay the Spire식 경로 시스템 — 5열 경로에서 매 노드마다 낚시/명당/어구점/휴식/보스 중 선택
  • 3단계 캐스팅 — CAST(파워 게이지) → BITE(타이밍) → REEL(텐션 게이지) 미니게임
  • 19종 낚시 비법 카드 — 점수형 12종 + 유틸리티 7종, 등급(common~legendary), 레벨업(Lv.1~3), 진화 카드 4종
  • 13종 어종 도감 — 망둥어, 우럭, 농어, 참치, 황금잉어 등 스테이지별 출현
  • 보스 방해 규칙 4종 — 야간 조업, 급류, 잔챙이 습격, 물때 마감
  • 명성 기반 영구 해금 — 6종 퍽(미끼 보너스, 시작 카드, 도감 보너스 등)
  • 온보딩 튜토리얼 — 첫 플레이는 손맛만, 2판부터 본격 카드/경로 시스템

🎨 그래픽 & 사운드

  • 13종 어종 일러스트(투명 배경, flat-vector 페인팅 스타일)
  • 스테이지별 시간대 분위기(낮/노을/밤)
  • 전설(tier4) 어종 포획 시 골드 화면 플래시 + 히어로 배너 + 화면 흔들림
  • Web Audio 사인파 합성 사운드 13종 (cast, bite, hook, reel, catch, perfect, evolve 등)
  • 점수 폭발 등급화 — 100/200/500점 구간별 시각 피드백

📜 게임 디자인

정본 디자인 문서는 게임과 함께 배포됩니다. v0.7의 핵심 시스템:

  • 경로 시스템: 시작 → 1열 → 2열 → 3열 → 보스, 5노드 통과, 슬레이 더 스파이어식 4열 확장
  • 3스테이지 누적 점수: 동네 방파제(30점) → 갯바위(80점) → 먼바다(180점)
  • REEL 난이도 곡선: 초록존이 스테이지별로 좁아지고 저항이 강해짐
  • 미끼 경제: 16개 시작, 휴식 +6, 어구점에서 구매
  • 리롤 토큰: 시작 2개, 보스 클리어/퍼펙트/신규어종 보너스
  • 시그니처: 5슬롯 가득 채우면 모든 점수 ×1.15
